브라우저 내장 객체(BOM:Browser Object Model)

브라우저 내장객체

Window 객체
브라우저의 창을 표현하는 객체다.
자바스크립트의 전역(Global)객체다.
script 태그 레벨에서 만들어지는 모든 변수, 모든 객체, 모든 함수를 자동으로 Window객체의 멤버가 된다.
Window객체를 제외한 다른 BOM객체는 모두 Window객체의 멤버다.
일반적으로 함수내에서 사용하는 this는 Window객체를 가르킨다.(객체내에 정의된 메소드는 예외다.)
자바스크립트에서는 window라는 참조변수로 접근할 수 있다.
console, alert, prompt, parseInt, parseFloat 등도 window객체의 멤버다.
script 태그 내부가 곧 window 객체의 영역이라고 생각하면 된다.
Document 객체
HTML 문서를 표현하는 객체다.
DOM(문서객체모델:Document Object Model)객체가 저장되어 있다.
DOM을 렌더링하면 브라우저에 컨텐츠가 표시된다.
DOM을 조작하면 브라우저 표시되는 컨텐츠를 조작할 수 있다.
W3C에서는 DOM을 조작하는 메소드를 표준화하였다.
Location 객체
브라우저의 현재 표시하고 있는 페이지에 대한 주소정보를 표현하는 객체다.
location.href의 값을 조회하면 현재 페이지의 주소를 조회할 수 있다.
location.href의 값을 변경하면 변경된 주소로 이동할 수 있다.
History 객체
브라우저의 페이지 방문기록 정보를 가지고 있는 객체다.
history.back()을 실행하면 브라우저의 뒤로가기 버튼을 누른 것과 같이 이전 페이지로 이동한다.
history.forward()을 실행하면 브라우저의 앞으로가기 버튼을 누른 것과 같이 다음 페이지로 이동한다.
Navigator 객체
브라우저에 대한 정보를 표현하는 객체다.
최신 자바스크립트 라이브러리들은 전부 크로스 브라우징(브라우저의 종류에 상관없이 동일한 동작을 하는 것)을 지원하기 때문에 브라우저의 정보를 조회할 필요가 없다.
Screen 객체
사용자의 모니터에 대한 정보를 표현하는 객체다.
