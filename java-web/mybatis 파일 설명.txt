src/main/resources/log4j.properties
	- 로그 출력 메세지 설정파일
		* mybatis 실행시 현재 실행상황을 출력

src/main/resources/mybatis/mybatis-config.xml
	- mybatis 환경설정파일
		* 데이터베이스 연결정보 설정
		* 사용자정의 매퍼파일 등록
		* mybatis 실행시 동작방식 설정

src/main/resources/mybatis/mappers/products.xml
	- mybatis 매퍼파일 * n개 (보통 table 갯수만큼 있음)
		* SQL 매핑정보를 설정

mybatis의 주요 객체
	SqlSession
		- SQL 매핑 구문을 실행한다.
		- SQL 매핑 구문을 실행할 때마다 생성하고, 실행이 완료되면 폐기한다.
		- 주요 메소드
			*[]안에 있는 건 생략 가능하단 의미
			T 	selectOne(String id[, Object parameter])
			List<E> selectList(String id[, Object parameter])
			int	insert(String id [, Object parameter])
			int	update(String id [, Object parameter])
			int	delete(String id [, Object parameter])
	SqlSessionFactory
		- SqlSession객체를 생성하는 객체다.
		- 한 개만 만들어서 애플리케이션이 실행되는 동안 계속 사용한다.
		- 주요 메소드
			SqlSession openSession()
			SqlSession openSession(boolean autoCommit)
	SqlSessionFactoryBuilder
		- mybatis 환경설정파일(mybatis-config.xml)을 로드해서 SqlSessionFactory를 생성한다.
		- 주요 메소드
			SqlSessionFactory build(InputStream in)
